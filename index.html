Tableau

This project aimed to conduct a comprehensive analysis of the camera and drone market. The objective was to increase our market share each year and present the final results to stakeholders at the end of 15 years.

To achieve this, we collected data for all years and organized it coherently in an Excel spreadsheet. After cleaning and verifying the data, we conducted an in-depth analysis using Tableau.

Power BI

This project involved analyzing the responses of over 4000 American participants who took part in a blind coffee taste test. The aim was to understand the preferences of the audience and determine which coffee was rated the best.

To ensure the accuracy of the data, I used Power Query to clean and organize it logically. I renamed some columns and words, trimmed and sorted certain value columns, and normalized three columns with "true" and "false" values to simplify and expedite data exploration. I also developed graphic analysis and chose less complex charts, such as pies and bars.

In the middle of the project, I decided to use the decomposition tree as an option for more in-depth analysis to obtain answers to a logical sequence of questions. I explored the graphs provided by Microsoft and uploaded them from the store to Power BI. If you need to answer a logical sequence of questions, this is a good solution for your analysis.

Finally, I utilized Figma to create an aesthetically pleasing layout for the project.

Excel

This project aimed to analyze bike sales across different regions and identify the primary audience for this mode of transportation. To present the data in a more understandable manner, I built a dashboard in Excel that included filters and graphs. To filter the data more effectively and handle the large amount of data, I utilized the slicer tool, which was created using a Pivot Table.

R

The aim of this project was to analyze the real estate market in Vancouver Downtown. I focused on redoing the Excel analysis with a different perspective. To achieve this, I applied some basic R knowledge and recreated the graphs. Using R for this kind of project can be more laborious than using Excel. For instance, exploring pivot tables, boxplots, and histograms is less intuitive and requires more coding and skills.
